第一次找的那个教程是bing搜索结果中首个知乎文章 Github学生认证及学生包保姆级申请指南 - 知乎，但根本没提多因素认证的事情，同时也使用了代理工具，被秒拒。我在第二次修改的时候给他点了个反对（其实也应该给bing也点一个负面反馈），很离谱啊朋友们。

  • You must configure your browser and operating system to allow location access. You may not use a VPN.   • Please secure your GitHub account with two-factor authentication. You may need to log out and log back in to GitHub before reapplying.

Please do not use a variation of your name or a nickname. Once you have updated your profile information log out and log back into GitHub before re-applying.
