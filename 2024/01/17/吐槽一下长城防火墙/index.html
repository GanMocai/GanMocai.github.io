吐槽一下长城防火墙
最近更新：2024-01-17   |   字数总计：387   |   阅读估时：1分钟   |   阅读量：次

回家后，没法裸连 github ，导致出现修改博客样式后部署出了一些问题，通过 google 又花了几十分钟解决，但可能解决时又引入新的隐患……故而想来吐槽一下：中国未免太不自信了！（用typora编辑时又出问题了，不过有上一次的经验，立即进入源代码模式编辑，也不耽误吐槽）

就我目前使用 github 的经历，我不认为封禁 github 有带来什么实际的好处。为了扶持本土的码云？别逗了，做得太差劲了，不太能成为百度/腾讯之流【本意是好的，可惜没能妥善考虑执行过程，不管本意如何，这一招做得不行】。预防颠覆？倒也没必要吧， github 上的更偏向技术，当然，也有一些历史存档，但因此而封禁未免太不自信！【未免太不自信，中国目前的太多操作如此：在封号禁言方面做了很多——和外交部发言人批评的对象倒是一般黑了。当然，也是因为中国共产党明白话语权的重要性以及“人民群众的眼睛是雪亮的”只是一句政治口号罢了……】

（话说我是真不行，科二练得太差了，寒假可能过不了，练的时候脑子不会动，脑子动的时候又回家了，没法练习-复盘-练习这样循环。说到底，这和我动手能力差同出一辙。）
